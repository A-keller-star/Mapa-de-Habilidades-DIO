# Mapa-de-Habilidades-DIO
Repositório criado para organizar o Mapa de Habilidades desenvolvido no curso da DIO. Contém as três etapas: inventário de habilidades, análise de mercado e caminhos de monetização. Inclui exemplos práticos, portfólio e README explicativo para apoiar a construção de uma marca pessoal e oportunidades profissionais.
